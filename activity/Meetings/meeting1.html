Potrace - piece of software that tries to derive vector representation of arbitrary image

Basic Idea:

-Android app that reads an image and with moving your finger along the image noises/vibrations occur based on images
-Intended to be used primarily for reading diagrams and graphs.
-Capable of reading short messages, and figures and outputting unique noises/vibrations

Key Ideas:

-Say you have an image too complicated for a blind person (like a technical diagram in EE380)
-Be able to read an image
-Use frequency of a note to indicate color
-When you hit an edge make a little click sound
-When there’s text in an image, run an OCR algorithm to read out text
-Being able to read the labeling in an axis since there’s no box
-If there’s something that looks like text but not in a boxed object, be able to make some kind of assumption in order to read the text
-If there’s text near some line but not in a box, play sounds at certain scenario
-You want everything to not require any network connections. All OCR should be local.

Hardware/Software Specifications

-Able to work on a cheap $40 android tablet (7 inches)
-Any brand will work.
-dual core / quad core
-Try to avoid requiring certain OS versions if possible
