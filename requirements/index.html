Requirements

    We will be using Android Studio (1.0.1) to create an Android app that allows visually impaired people to understand diagrams through the use of sounds and vibrations. To achieve this task, we will be following the specifications below.

HardWare / Software Specifications:

  • App is capable of running on the cheapest 7 inch modern tablets that exist.
  • Tablets used will be dual core / quad core.
  • A maximum of 1GB of RAM will be available in the tablets that are used.
  • Android 4.4 (Kitkat) will be the operating system used for testing.

Shape Detection

  • When reading diagrams, the application will detect any rectangles in the diagram whenever the user moves their finger to its location. Different sounds will occur when touching a specific rectangle on the diagram.
  • All other shapes that the diagram may have are currently not being considered. Detecting other shapes will be implemented if time permits.

OCR

  • Upon touching the tablet, an OCR will be ran to check if there exists text inside of a rectangle border. If text is found then the letters will be read out to the user.
  • When the OCR notices text, a beeping noise will be made that will notify the user that the OCR has found text. This lets the user know that they need to stop moving their finger so that the OCR can read the text.

Reading Images

  • The application will accept images from Android's gallery.
  • The app allows users to load in images directly from the gallery.
