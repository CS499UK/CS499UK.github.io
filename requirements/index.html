Basic Idea:

-Android app that reads an image and with moving your finger along the image noises/vibrations occur based on images
-Intended to be used primarily for reading diagrams and graphs.
-Capable of reading short messages, and figures and outputting unique noises/vibrations.

HardWare / Software Specifications:

-Able to work on a cheap $40 android tablet (7 inches)
-Any brand will work.
-dual core / quad core
-Try to avoid requiring certain OS versions if possible

Iteration 1

Goals

  • Gain a better understanding of gesture controls
  • Achieve basic functionality in relation to getting color at a particular point
  • Gain experience using Android libraries

Strategy

  • Accept very strict dimmensioned pictures (pdf)
  • Color code diagram
  • Implement array of colors and corresponding labels
  • Hard code sound confirmation if finger touches boundary line of object
  • Hard code voice confirmation of object finger is in/on
