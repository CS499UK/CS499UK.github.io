Requirements

    We will be using Android Studio to create an Android app that allows visually impaired people to understand diagrams through the use of sounds and vibrations. To achieve this task, we will be following the specifications below.

HardWare / Software Specifications:

  • App is capable of running on the cheapest 7 inch modern tablets that exist.
  • Tablets used will be dual core / quad core.
  • A maximum of 1GB of RAM will be available in the tablets that are used.
  • Android 4.4 (Kitkat) will be the operating system used for testing.

Color Reading

  • When reading diagrams, the application will read the pixel values based on where the user touches the tablet. Different sounds will occur when touching a specific color, or moving to a new color on the diagram.
  • When the user runs into the pixel value for black, a unique sound and vibration will occur. This will notify the user that they have hit the color black and have likely touched a border.

OCR

  • Upon touching the color black, an OCR algorithm will be ran to check if there exists text inside of a border. If text is found then the letters will be read out to the user.

Reading Images

  • The application will accept PDF images. In the future we will integrate more features that allow the user to read other formats.
